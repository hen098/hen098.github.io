실패 케이스

  1. 존재하지 않는 사용자
    • InternalAuthenticationServiceException
  2. 유효하지 않은 사용자
    • DisabledException
  3. 비밀번호 실패
    • BadCredentialsException
    • 비밀번호 실패횟수 + 1
    • 실패 기준 횟수 초과시에 계정잠금 (lock 컬럼을 true로)
  4. 계정잠김
    • LockedException
    • front 에서 받아서 캡챠화면으로 이동
